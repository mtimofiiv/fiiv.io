Cache Horse

Building data-driven applications is fun and challenging at the same time. One of the challenges I have encountered again and again in building these types of applications concern grabbing the data itself.

Cache Horse is a cache-as-a-service. One simple API endpoint gets you a cache on any API endpoint out there - and you can batch the requests together to reduce your HTTP requests!

Status: live
Technologies used: Ruby on Rails, Sqlite, Redis, Docker
